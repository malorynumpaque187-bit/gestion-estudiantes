🎓 API REST - Gestión de Estudiantes

Versión: 1.0 | Desplegado con: AWS CodePipeline + S3 | Estudiante: Malory Numpaque

Endpoints disponibles:

GET/students — Listar todos los estudiantes
POST/students — Crear un nuevo estudiante
GET/students/{id} — Obtener estudiante por ID
GET/salud — Estado del servicio

Pipeline CI/CD activo — los cambios se despliegan automáticamente desde GitHub.
